HEY IF YOU SEE THIS KNOW THAT IT IS NOT YET READY DESPITE NOT ACTIVELY BEING DISABLED

"On The Fly" Collections are an alternative way to create collections designed to integrate collection functionality throughout the site.

By enabling On The Fly Collections, most pages on the Museum will have form controls along the top of the page that allow you to quickly add whatever file is being viewed to a collection.

The "X" button on the right side of the header will disable the functionality until it is activated again.

A drop down menu will allow specifying a collection to add to, with the ability to create a new collection as well. Newly created collections in this manner will have their visibility set to PRIVATE.
